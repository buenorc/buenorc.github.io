Grid generator

This program generates the Orthogonal curvilinear grid file (.grd) that fits into the landboundary (shapefile) to be used in the Delft3D flow (available with a GUI). This algorithm is implemented in Python

Decomposition model (Interwave Analyzer)

This program uses outputs from Interwave Analyzer and measured horizontal velocities to obtain the horizontal velocity associated with each internal seiche mode. This algorithm is implemented in Python (no GUI available)
